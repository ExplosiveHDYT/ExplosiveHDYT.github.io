Leaflet

Google-maps-style pan & zoom demo

This is a simple demo of panning and zooming a large image in a Webflow site, using leaflet.js.

Because it's javascript-based, it is only viewable in the published site.

In this case, it's a single untiled image, with no real coordinate system. See the /head and /body custom code areas for implementation details.
